# 所得税計算アプリ

このプロジェクトは、ブラウザ上で簡易的に所得税額を概算するためのサンプルアプリです。

## 概要

ユーザーは各種の所得（利子、配当、不動産、事業、給与、退職、山林、譲渡、一時、雑）を入力できます。合算した総所得から所得控除を差し引き、課税所得金額に対して累進税率を適用して所得税を算出します。最後に税額控除を引いて申告納税額（概算）を表示します。

## 使い方

1. `incomeTax/index.html` をブラウザで開きます（ファイルをダブルクリックするか、ブラウザの「ファイルを開く」機能を使用します）。
2. 各所得欄に金額を入力します（未入力は 0 と扱われます）。
3. 「所得控除 合計」に控除合計額を入力し、「税額控除 合計」に税額控除の合計額を入力します。
4. 「計算」ボタンを押すと、課税標準・課税所得・所得税額（概算）・申告納税額が表示されます。

## 注意（重要）

- このツールは教育・参考用の「簡易モデル」です。実際の税務申告は税法・通知・各種控除の詳細が関わるため、正式な税額計算には専門家または国税庁の資料を参照してください。
- 課税のための各種調整（給与所得控除の自動計算、退職所得の特例、分離課税扱いの譲渡所得など）や住民税・復興特別税は考慮していません。
- 税率表は説明目的の簡易的な累進税率を使用しています（参考値）。

## 開発メモ

- `app.js` に計算ロジックを実装しています。関数 `computeTax` は累進税率に基づき概算の税額を計算します。
- 将来的な改善: 給与所得控除の自動反映、各種所得の別表処理、住民税の計算、税務上の特例や控除の詳細追加。
