# ToDoアプリ

このリポジトリはシンプルな ToDo アプリのサンプルです。ブラウザのみで動作し、追加・完了・削除の基本操作と、ブラウザの localStorage による永続化を備えています。

## ファイル構成

- `index.html`: ユーザーインターフェース（入力欄、追加ボタン、タスクリスト）
- `todo-list.js`: タスクの追加・表示・完了切替・削除、localStorage への保存を実装したスクリプト

## 使い方

1. このフォルダ (`todo-app`) 内の `index.html` をブラウザで開きます。ファイルをダブルクリックするか、ブラウザで開いてください。
2. テキストボックスにタスクを入力して「追加」ボタンをクリック、または Enter キーでタスクを追加します。
3. 追加したタスクは一覧に表示されます。チェックボックスで完了の切替、削除ボタンでタスクを削除できます。

4. ページ上部の「削除」ボタンを押すと、チェックが入ったタスクをまとめて削除できます。

5. タスクの本文を編集するには、一覧のタスクテキストをクリックしてください。入力欄と「編集」ボタンが表示されます。編集後に「編集」ボタンを押すか Enter キーで確定します。

## 永続化

タスクはブラウザの `localStorage` に保存されます。ページを閉じても次回開いたときに復元されます（同一ブラウザ・同一マシン環境）。

## 開発メモ

- `todo-list.js` は即時実行関数 (IIFE) 内で動作します。tasks は配列で管理され、`STORAGE_KEY = 'todo_app_tasks_v1'` を使って保存しています。
- 将来的な拡張案: 編集機能、フィルタ（全て/未完了/完了）、drag-and-drop 並べ替え、同期バックエンド。

# ToDoアプリ

このリポジトリはシンプルな ToDo アプリのサンプルです。ブラウザのみで動作し、追加・完了・削除の基本操作と、ブラウザの localStorage による永続化を備えています。

## ファイル構成

- `index.html`: ユーザーインターフェース（入力欄、追加ボタン、タスクリスト）
- `todo-list.js`: タスクの追加・表示・完了/選択・削除、localStorage への保存を実装したスクリプト

## 新しい機能: 「完了」ボタン

ページ上部に「完了」ボタンを追加しました。

- タスクの左のチェックボックスは「選択」を表します（完了対象や削除対象の選択）。
- チェックを入れてからページ上部の「完了」ボタンを押すと、選択されたタスクが完了扱いになり、一覧上で取り消し線（strike-through）が適用されます。
- 「削除」ボタンは選択されたタスクをまとめて削除します。

## 使い方

1. このフォルダ (`todo-app`) 内の `index.html` をブラウザで開きます。
2. テキストボックスにタスクを入力して「追加」ボタンをクリック、または Enter キーでタスクを追加します。
3. タスク一覧の左のチェックボックスを使ってタスクを選択します。
4. 選択したタスクを完了にするにはページ上部の「完了」ボタンを押します。完了済みのタスクには取り消し線が表示されます。
5. 選択したタスクを削除するにはページ上部の「削除」ボタンを押します。

## 永続化

タスクはブラウザの `localStorage` に保存されます。ページを閉じても次回開いたときに復元されます（同一ブラウザ・同一マシン環境）。

## 開発メモ

- `todo-list.js` は即時実行関数 (IIFE) 内で動作します。tasks は配列で管理され、`STORAGE_KEY = 'todo_app_tasks_v1'` を使って保存しています。
- `task` オブジェクトは少なくとも `{ id, text, done, selected }` のプロパティを持ちます。

4. 選択したタスクを削除するには「削除」ボタンを押します。
